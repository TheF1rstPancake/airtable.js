Query(table, params)

new Query(table, params)

Builds a query object. Does not actually fetch any data until either Query.eachPage, Query.firstPage, or Query.all is called.

Parameters:
Name Type Description
table Table

Airtable table object representing the table we want to query against
params Object

an associative array containing the different URL parameters for the query. See https://airtable.com/api for more details.

You must provide the pageCallback to perform actions on each page. This funciton will be called after each page has been fetched.

Type Definitions

done(error, results)

Callback function for the final results or error of a query

Parameters:
Name Type Description
error Error

if the query failed, error will represent the error that occured. If the query succeeded, error will be null
results null | Array.<Record>

If there is an error, this will be null. If successful and using Query.all or Query.firstPage, this will be a list of records matching your query. If using Query.eachPage this will be null.
