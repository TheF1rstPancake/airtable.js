Type Definitions

AirtableConfig

Properties:
Name Type Description
apiKey String

your Airtable API key. By default, the SDK pulls this from the AIRTABLE_API_KEY environment variable
endpointUrl String

the base URL for the Airtable API. Defaults to https://api.airtable.com
apiVersion String

the API version you want to use. Airtable's current API Version is "v0".
allowUnauthorizedSsl Boolean

If not false, the server certificate is verified against the list of supplied CAs. An error occurs if the verification fails. Otherwise, calls will succeed even if SSL verification fails (for example, when using a self-signed certificate). Defaults to false.
noRetryIfRateLimited Boolean

If not false, you are responsible for managing errors with errorCode 429 when you exceed the API rate limit. Otherwise, the Airtable SDK will manage the timeout logic. Defaults to false (Airtable SDK manages timeouts by default).
requestTimeout Number

number of miliseconds to wait before timing out the request. Defaults to 300,000 (5 minutes)
